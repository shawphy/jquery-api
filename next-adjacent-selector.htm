prev + next

选择所有紧跟在 "prev" 后面的，匹配 "next" 选择器的元素。

  • 1.0 新增prev + next

    prev (Selector) 任何有效的选择器。
    next (Selector) 用于筛选紧跟在后面的元素的选择器。

非常重要的一点是，(prev + next) 和 (prev ~ siblings) 所选择到的元素，必须是在同一个父元素下的。

示例:

查找所有跟在 label 后面的 input 元素。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<form>

    <label>Name:</label>
    <input name="name" />
    <fieldset>
      <label>Newsletter:</label>

      <input name="newsletter" />
    </fieldset>
  </form>
  <input name="none" />

<script>

$("label + input").css("color", "blue").val("Labeled!")

</script>
</body>
</html>

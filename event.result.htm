返回值:Objectevent.result

当前事件最后触发的那个处理函数的返回值，除非值是 undefined 。

  • 1.3 新增event.result

这个属性经常用于获取自定义事件的前一个返回值。

示例:

显示前一个事件处理函数的返回值。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>


<button>display event.result</button>
<p></p>


<script>


$("button").click(function(event) {
  return "hey";
});
$("button").click(function(event) {
  $("p").html( event.result );
});  


</script>
</body>
</html>

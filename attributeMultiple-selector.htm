[attributeFilter1][attributeFilter2][attributeFilterN]

匹配满足所有这些属性选择器的元素。

  • 1.0 新增[attributeFilter1][attributeFilter2][attributeFilterN]

    attributeFilter1 (Selector) 属性过滤器。
    attributeFilter2 (Selector) 另一个属性过滤器，用于进一步减少选择的元素。
    attributeFilterN (Selector) 可选参数，如果需要，可以跟更多的属性选择器。

示例:

查找那些有 id 属性，并且 name 属性以 man 结尾的输入框，并设置文本值。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<input id="man-news" name="man-news" />

  <input name="milkman" />
  <input id="letterman" name="new-letterman" />
  <input name="newmilk" />

<script>

$('input[id][name$="man"]').val('only this one');

</script>
</body>
</html>

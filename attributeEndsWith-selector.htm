[attribute$="value"]

选择那些属性值是以给定字符串结尾的元素。这里对大小写敏感。

  • 1.0 新增[attribute$="value"]

    attribute (String) 属性名。
    value (String) 属性值，引号是强制性的。

示例:

查找所有 name 属性是以 'letter' 开头的 input 元素，并放置一些文本。

<!DOCTYPE html>
<html>
<head>
<script src="jquery.min.js"></script>
</head>
<body>

<input name="newsletter" />

  <input name="milkman" />
  <input name="jobletter" />

<script>

$('input[name$="letter"]').val('a letter');

</script>
</body>
</html>
